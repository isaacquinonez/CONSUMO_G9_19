G9_19 | CONSUMO DE API's RES

— PAGOS: —

  • Este formulario contiene toda la información de los pagos realizados por los diversos clientes que han adquirido un producto o servicio brindado de nuestra parte. Esta información puede ser utilizada con fines informativos, como también puede ser modificada (solo personal autorizado) en caso de ser requerido.

  • Dentro del presente formulario podrán realizar las siguientes funciones:
    • PAGO.html: Aquí se encuentra todo el código de diseño básico que se visualiza en la página web.
    • PAGO.js: Aquí se encuentra todo el script con las funciones CRUD necesarias para poder hacer funcional la página web del html.

  • Dentro del presente formulario podras realizar las siguientes opciones:
    • AGREGAR: Con el botón de Agregar podrás ingresar nuevos pagos realizados.
    • ACTUALIZAR: Con el botón de Editar podrás hacer cambios en los pagos realizados dentro del registro.
    • ELIMINAR: Con el botón de Eliminar podrás borrar un registro de pago realizado anteriormente, esto en caso de ser necesario.
